Procédé pour obtenir le rôle Discord associé à votre compte Root-Me

Faire vérifier votre compte par le bot Root-Me#3551 en lui envoyant la commande par message privé :

!verify VotreApiKey

Vous trouverez votre API key dans vos paramètres de compte.

En fonction de votre statut sur Root-Me (visiteur, membre de l’association, premium...), vous obtiendrez le rôle correspondant sur Discord, vous ouvrant l’accès à certains canaux et permissions supplémentaires.

Attention : cette commande contient votre clé d’API privée, elle ne doit pas être envoyée sur un canal public du serveur Discord mais bien en message privé au bot Root-Me#3551. Ce bot est mentionné sur les canaux de bienvenue.

Règles du serveur

Respecter la charte d’utilisation de Discord Community. Avoir une attitude correcte sur le serveur.
Respecter les mentions légales et les conditions générales d’utilisation de Root-Me. Pas de cheat, leak, ou demandes illégales.
Pas de spam. Toute publicité pour un service externe à des fins financières ou malveillantes est interdite.
Soyez identifié avec votre propre compte Root-Me. Votre pseudo Discord doit être le même que sur Root-Me et vous devez utiliser votre propre clé API pour la vérification via @Root-Me#3551.

À noter : vous pouvez donner l’identifiant que vous voulez à votre compte Discord, votre pseudonyme sera automatiquement modifié sur le serveur Root-Me pour correspondre à votre pseudonyme du site web.
